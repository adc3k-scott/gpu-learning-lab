## What We Do

We build self-contained AI computing pods inside modified shipping containers. The assembly process involves cutting steel, spray foam insulation, electrical wiring, plumbing, and installing computing hardware. We then test each unit before shipping to customer sites. This is light manufacturing — no heavy industrial processes, no hazardous waste, no significant noise.

## Building Requirements

### MUST HAVE — Non-Negotiable

| Requirement | Specification | Why |
|-------------|---------------|-----|
| **Clear span floor** | 10,000 SF minimum, 15,000+ preferred | Assembly stations need open floor, no columns |
| **Ceiling height** | 20 ft minimum clear | Overhead crane + container clearance |
| **Truck access** | Drive-in door, 12ft W × 14ft H minimum | 20ft containers delivered by flatbed |
| **Loading dock** | 1 truck bay minimum | Material receiving + finished pod shipping |
| **3-phase power** | 200A minimum at 480V, 400A preferred | Welding, plasma cutting, and test equipment |
| **Concrete floor** | 6" slab or better | Must support 10,000 lb point loads from containers |
| **Industrial zoning** | M-1 or M-2 (manufacturing permitted) | We are building products, not warehousing |
| **Parking** | 15+ spaces | 10-12 staff + visitors |

### STRONGLY PREFERRED

| Feature | Value to ADC |
|---------|-------------|
| Existing overhead crane (5-10 ton) | Saves $45-120K and 4 weeks of install time |
| 2 loading docks (separate receive/ship) | Avoids schedule conflicts |
| 2 drive-in doors | Container in/out from both ends |
| Fenced yard (0.5-1 acre) | Outdoor container staging for 4-6 units |
| Small office buildout (200-400 SF) | Operations manager + QA desk |
| Existing restrooms | Cheaper than new construction |
| Compressed air system | Saves $5K install |
| I-10 / I-49 access within 5 miles | Shipping efficiency |

### DEAL BREAKERS

- Residential zoning or mixed-use
- Ceiling height below 20 ft
- No 3-phase power available (or upgrade cost >$50K)
- Flood zone without elevated pad
- Environmental contamination / active remediation
- Shared space / multi-tenant with noise restrictions
