## 4-Layer Mini Grid (Willow Glen ONLY — differs from standard 3-layer)
```
Layer 1A: Solar Array + Battery         RENEWABLE · daytime primary · 30-40% offset
Layer 1B: Nat Gas (Henry Hub gensets)   LOW-CARBON · 24/7 baseload · cheapest fuel
Layer 2:  Grid (CLECO/Entergy)          BACKUP
Layer 3:  Biodiesel Gensets             EMERGENCY · renewable · existing tank storage
```

## Renewable Energy Roadmap
1. **Day One**: Solar + battery (Layer 1A) + nat gas gensets (Layer 1B)
2. **12-18 months**: Green hydrogen production via electrolysis
   - Solar excess powers electrolyzers
   - Hydrogen stored in EXISTING tank farm (already rated for gas)
   - Burned in modified generators for 24/7 zero-carbon power
   - Ship terminal can receive green hydrogen from Gulf Coast production
3. **Future**: Renewable Natural Gas (RNG) via pipeline — carbon-neutral methane
4. **Emergency**: Biodiesel in existing tanks for Layer 3 gensets (B100 compatible)

## Why Willow Glen Is Uniquely Positioned for Renewables
- Tank farm = hydrogen/fuel storage already built (no new construction)
- Ship terminal = marine fuel delivery (hydrogen, biodiesel, RNG by barge)
- Pipeline connections = RNG delivery without truck transport
- Grid interconnect = can SELL excess power back to grid (revenue)
- No other AI facility site in the US has this combination
