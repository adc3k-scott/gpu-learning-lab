# Claude Design Review - Workflow & Process

## Overview
Established reusable workflow for sending projects to Claude Design for visual enhancement while preserving structure and functionality.

## When to Use This
- Resume/portfolio needs visual modernization
- Content is solid but presentation needs polish
- Want professional design enhancement without restructuring
- Need to maintain all existing functionality and content

## Process Steps

### Phase 1: Prepare Project
1. Ensure project is complete and functional
2. Test all features, navigation, media
3. Document what should NOT change (hard constraints)
4. Identify enhancement goals (modern look, better spacing, etc.)

### Phase 2: Create Design Package
```powershell
# Create dedicated folder
mkdir claude-design-review -Force

# Copy main files
Copy-Item project-file.html "claude-design-review\"
Copy-Item supporting-assets "claude-design-review\" -Recurse
```

### Phase 3: Create Design Brief
Create `DESIGN_BRIEF.md` with:
- **Overview** - What the project is
- **Current State** - What's working, what needs help
- **Design Goals** - What improvements wanted
- **Hard Constraints** - What MUST NOT change
- **Technical Details** - File structure, tech stack
- **Success Criteria** - How to measure success
- **Delivery Specs** - Format expectations

### Phase 4: Submit to Claude Design
Share the `claude-design-review` folder with:
- **What to say:** "Please enhance the visual design while maintaining the structure outlined in DESIGN_BRIEF.md"
- **What Claude Design will do:** Update CSS, refine typography, improve layouts, add polish
- **What they'll return:** Enhanced HTML/CSS with all structure preserved

### Phase 5: Implementation
- Claude Design returns enhanced files
- Merge updated files into your project
- Test all functionality remains intact
- Deploy/publish

## Key Success Factors
✅ Clear design brief prevents miscommunication  
✅ Self-contained package makes handoff easy  
✅ Hard constraints protect your core structure  
✅ Success criteria set clear expectations  
✅ Reusable template saves setup time  

## Tools & Files
- **DESIGN_BRIEF.md** — Communication document
- **Project files** — Main deliverables
- **Assets folder** — All supporting media
- **README** (optional) — Quick reference

## Example: Scott Resume Enhancement
- **Package:** `/claude-design-review/`
- **Main file:** `scott_tomsu_resume.html`
- **Assets:** `Nvidia-certs/`
- **Brief:** `DESIGN_BRIEF.md`
- **Goal:** Modern visual polish, premium appearance
- **Constraints:** Keep all 16 slides, navigation, media
